# Vidriería - React + Vite

## Instalación

1. `npm install`
2. `npm run dev`

Abre http://localhost:5173

## Notas
- Toda la data se guarda en `localStorage` (claves: `vid_prod`, `vid_sales`, `vid_entries`).
- Si quieres resetear datos borra esas claves desde DevTools -> Application -> Local Storage.
