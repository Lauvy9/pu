📊 PROCESAMIENTO DE CONSULTA

  ENTRADA: "holaa que vendimso mas???"
  
     ↓ NORMALIZACIÓN
     "hola que vendimos mas"
  
     ↓ DETECCIÓN DE INTENCIÓN
     intent: "topSales", hasGreeting: true
  
     ↓ EXTRACCIÓN DE DATOS
     topProducts: [{name, salesCount}, ...]
  
     ↓ GENERACIÓN DE RESPUESTA
     "¡Hola! 👋 Estoy acá para ayudarte...
      📈 **Productos más vendidos:**
      1. Cerveza (25 ventas)
      2. Vino (18 ventas)
      3. Whisky (12 ventas)"
  
     ↓ OUTPUT
     response, intent, params, metadata

🚀 CÓMO USAR

  1. Ingresa a tu aplicación
  
  2. Abre el chat flotante 🤖 en esquina inferior derecha
  
  3. Escribe cualquier cosa (incluso con errores):
     • "hola que se vendio mas"
     • "vendimso hoy cuanto"
     • "stock bajo"
     • "recomiendame para cliente 123"
     • "mostrar productos"

  5. Si preguntas 3+ veces lo mismo, obtendrás tips adicionales

🔧 TÉCNICAS IMPLEMENTADAS

  Algoritmo de Levenshtein Distance
  └─ Para calcular similitud entre palabras
  └─ Detecta palabras malescritas automáticamente
  └─ Threshold: 0.75 de similitud

  Feedback Learning
  └─ Rastrea consultas frecuentes en sesión
  └─ A partir de 3 consultas = respuesta mejorada
  └─ Sistema simple pero efectivo

  Normalización de Texto
  └─ Pipeline de: minúsculas → signos → duplicaciones → correcciones
  └─ Mantiene tildes (áéíóú, ñ)
  └─ Stopwords identificados y removidos

  Extracción de Intención
  └─ Búsqueda exacta primero
  └─ Fuzzy matching como fallback
  └─ Detección de múltiples intenciones
